# -SCT_WD_1
The objective of this task was to create an interactive navigation menu that changes its style when the user scrolls the webpage or hovers over menu items. The navigation menu should also remain fixed at the top of the page so that it is always visible to the user

In this project, I developed a responsive navigation menu using HTML, CSS, and JavaScript. The navigation bar remains fixed at the top of the webpage and changes its appearance when the user scrolls down the page.

Additionally, hover effects were applied to the navigation menu items to make the interface more interactive and visually appealing. The design is responsive and works well on different screen sizes.

Features
Fixed navigation bar that stays visible while scrolling
Navigation bar style changes when the page is scrolled
Hover effects on navigation menu items
Responsive design for different screen sizes
Smooth user interface interaction


Technologies Used
HTML5 – Used to structure the webpage
CSS3 – Used for styling and hover effects
JavaScript – Used to implement scroll-based interactions
